# Technical Review and Source Availability

## Public Repository

This repository contains public-facing submission materials, architecture documentation, workflow maps, selected module descriptions, screenshots, demo scripts, and project context.

## Proprietary Systems

Some implementation details are intentionally not included in this repository.

Excluded proprietary areas may include:

- private orchestration systems
- retrieval infrastructure
- graph generation systems
- warehouse tooling
- vector/corpus indexes
- Runtime-C internals
- Tracer internals
- private deployment configuration
- secrets, keys, wallets, and customer data

## Review Upon Request

Additional technical review is available for qualified reviewers.

Available upon request:

- architecture walkthrough
- live system demonstration
- private server inspection
- selected source review
- implementation explanation
- infrastructure discussion
- partnership or investment diligence

This project is presented as a working prototype and product foundation. The public repository is intentionally scoped for evaluation while protecting the internal development engine and proprietary architecture.
