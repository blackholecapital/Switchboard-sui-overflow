# Runtime-C, Retrieval, and Proprietary Tooling

Switchboard was developed with internal retrieval and resource-pack systems that assist with code discovery, architecture planning, and module assembly.

## Public Summary

The internal system is designed to:

- discover implementation patterns
- rank source candidates
- build resource packs
- generate metadata
- assist architecture planning
- support large-codebase navigation
- accelerate product development

## Proprietary Scope

The following are intentionally excluded from the public repository:

- Runtime-C internals
- Tracer asset warehouse
- private retrieval indexes
- vector stores
- graph generation pipelines
- resource-pack generation logic
- internal prompts and orchestration
- customer data
- private deployment infrastructure

## Review Availability

Technical walkthroughs and private inspection can be arranged for:

- hackathon judges
- Sui ecosystem teams
- technical partners
- investors
- serious collaborators

The public repo is intended to demonstrate the project, not expose the internal factory that produced it.
