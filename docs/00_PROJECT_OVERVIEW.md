# Switchboard OS Project Overview

Switchboard OS is a modular business operating system that brings CRM, billing, storage, treasury, documents, communications, workflow automation, and AI assistance into one connected workspace.

The project is designed around a business-first view of blockchain adoption. A business user should be able to create an invoice, trigger a payment, store records, create receipts, and view treasury activity without needing to manage blockchain details manually.

## Core Thesis

Blockchain infrastructure becomes useful to ordinary businesses when it is attached to familiar workflows:

- invoices
- payments
- receipts
- backups
- documents
- treasury records
- customer activity
- operational audit trails

Switchboard uses Sui, USDC payments, wallet-aware actions, attestation records, and storage workflows to make those events verifiable.

## Primary Modules

| Module | Purpose |
|---|---|
| Billing360 | Invoices, PayMe checkout, Stripe/Sui USDC payment surface |
| Storage360 | Backup controls, storage receipts, Walrus-oriented records, Sui receipt trail |
| Treasury360 | Wallet visibility, balances, payment context, treasury operations |
| Documents360 | Document workflows, document records, attestations |
| CRM Core | Contacts, companies, deals, tasks, records, sidecars |
| Automation360 | Workflow orchestration and operational automation |
| Comms360 | Communication layer and message-oriented workflows |
| Workspace360 | Scheduling, workspace tasks, operational utilities |
| AI Sidecars | Module-level AI support, context, guidance, and workflow assistance |
